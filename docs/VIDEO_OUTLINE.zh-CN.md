# 视频说明模板

## 视频标题
Alpha Radar | 基于 Binance Skills 的多维体检报告生成与 Square 发布工作流

## 视频简介
本项目是一个 OpenClaw 自定义 Skill。它不重写 Binance 官方数据接口，而是把官方 Skills 采集到的市场、代币、交易信号、合约审计与聪明钱数据整理为固定的日报结构，并可进一步生成适合 Binance Square 的发文草稿。

## 演示重点
1. 用 Binance 官方 Skills 做数据输入
2. 用 Alpha Radar 做结构化报告输出
3. 展示“市场主线 / 值得看名单 / 风险警报 / 聪明钱附录”的固定栏目
4. 展示 Square 文案预览
5. 展示 GitHub 仓库结构

## 建议镜头顺序
1. README
2. SKILL.md
3. sample-data.json
4. npm run report
5. npm run square
6. 生成结果
